# Roomly

Roomly is a travel listing web application inspired by Airbnb, built using Node.js and MongoDB.

# About

Roomly allows users to explore travel destinations listed by other users. Anyone can sign up, create their own listing with images, and leave reviews on others. The platform focuses on simplicity and ease of use.

# Features

- User authentication and authorization using Passport.js
- Create, edit and delete travel listings
- Upload listing images using Cloudinary
- Leave star ratings and written reviews on listings
- Only listing owners can edit or delete their own listings
- Only review authors can delete their own reviews
- Category filters like Trending, Mountains, Farms, Boats and more
- Responsive design for mobile and desktop
- Flash messages for success and error feedback
- Session stored in MongoDB using connect-mongo

# Tech Stack

- Node.js and Express.js for the backend
- MongoDB Atlas and Mongoose for the database
- EJS and Bootstrap 5 for the frontend
- Passport.js for authentication
- Cloudinary and Multer for image uploads
- Render for deployment
